# Итоговая проверочная работа.

### Задача: 
Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа. Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма. При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.

### Алгоритм решения:
1. Производим перебор значений исходного массива.
2. Проверяем каждое значение из массива на соответствие условию: ```длина строки меньше или равна трем.```
3. Если строка удовлетворяет условию, записываем значение в новый массив.
4. Повторяем пункты 2 и 3 до тех пор пока не достигнем конца данного массива.
5. Возвращаем новый заполненый массив как результат.

### Программа:
Для запуска программы, откройте терминал, перейдите в необходимую директорию и запустите команду:
```
dotnet run
```
**Введите массив, например:**
```
ok soccer 007 Markdown 77
```
**Пример вывода программы:**
```
ok 007 77
```
